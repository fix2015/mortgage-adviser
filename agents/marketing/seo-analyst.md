## On-Page SEO Checklist
For every page/blog post:
- [ ] Title tag: Primary keyword + brand, under 60 chars
- [ ] Meta description: Benefit-driven, includes keyword, under 155 chars
- [ ] H1: Contains primary keyword, unique per page
- [ ] H2s: Include secondary keywords naturally
- [ ] URL slug: Short, keyword-rich, hyphenated
- [ ] Internal links: 3-5 per post to calculator, landing page, related content
- [ ] External links: 1-2 authoritative sources (Bank of England, FCA, gov.uk)
- [ ] Image alt text: Descriptive, includes keyword where natural
- [ ] Schema markup: FAQ, HowTo, or Article schema
- [ ] Mobile-friendly: Passes Google mobile test
- [ ] Page speed: Under 3s load time
- [ ] Word count: 1,500+ for blog posts

## Technical SEO
- [ ] Sitemap.xml generated and submitted
- [ ] robots.txt configured correctly
- [ ] Canonical URLs set on all pages
- [ ] 301 redirects for any changed URLs
- [ ] No broken links (run monthly scan)
- [ ] SSL certificate active
- [ ] Core Web Vitals passing (LCP < 2.5s, FID < 100ms, CLS < 0.1)
- [ ] Structured data validated via Google Rich Results Test
